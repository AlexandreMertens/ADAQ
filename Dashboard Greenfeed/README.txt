Placer ce dossier à la racine de votre disque C:

Télécharger php avec ce lien : http://php.net/downloads.php
Selectionner le lien windows downloads de la dernière version stable.
Puis télécharger le Zip Non Thread qui correspond à votre version 64 bits ou 32 bits.
Créer un dossier nommé "phpresource" à la racine de votre disque C:.
Extrair le Zip à l'intérieur sans créer de dossier intermédiaire.
Placer également le document CRA-W dans le dossier phpresource.

Télécharger python avec le lien : https://www.python.org/downloads/
Une fois l'installation fini editer le script DashboardGF.bat qui se situe dans le dossier "Dashboard Greenfeed".
et modifier le chemin jusqu'a python.exe avec le chemin de votre ordinateur.

Exemple C:\Users\l.degreve\AppData\Local\Programs\Python\Python37-32
 
Faîtes de même avec le chemin jusqu'a votre navigateur.
Placer le fichier getGFdatabyD.py et getGFdatabyH.py dans le dossier qui contient python. 
Voila vous pouvez lancer getGFdata.bat
